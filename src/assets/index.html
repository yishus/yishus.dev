About

My name is Yishu (like a GitHub issue) and I'm a software engineer. In my day job I manipulate data and design APIs. My interest lies in tool design which has led me to explore areas such as programming language design and code bundling. I'm drawn to problems that reward taste and software that feels obvious in hindsight and slightly embarrassing to explain in detail.
